# Dewflow Lab

Personal project lab and technical review archive built with Astro, MDX, TypeScript, and plain CSS.

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Cloudflare Pages

- Build command: `npm run build`
- Build output directory: `dist`
- Production branch: `main`

Content directories:

- `src/content/projects/`: project review MDX
- `src/content/posts/`: posts and engineering notes MDX
- `public/images/`: screenshots, covers, and static assets
